# 📊 Quantitative Finance Model Comparison

This project implements and compares advanced stochastic models for stock price forecasting, combining mathematical finance with data-driven simulation techniques.

## 🧠 Models Implemented

- **Geometric Brownian Motion (GBM)**
- **Heston Stochastic Volatility Model**
- **GARCH-driven GBM**

## ⚙️ Methodology

- Financial time series analysis using historical market data  
- Monte Carlo simulation for price forecasting  
- Model comparison based on predictive behavior  

## 📈 Output

The system generates:
- Forecasted price paths  
- Comparison between models  
- Visualization against actual market data  

## 🛠️ Tech Stack

- Python  
- NumPy, Pandas  
- Matplotlib  
- ARCH (GARCH modeling)  
- yFinance
