# Quant Finance Model Comparison

This project compares multiple quantitative models for stock price forecasting:

- Geometric Brownian Motion (GBM)
- Heston Stochastic Volatility Model
- GARCH-driven GBM

## 📊 Features
- Monte Carlo Simulation
- Financial Time Series Analysis
- Model Comparison

## 🛠️ Tech Stack
- Python
- NumPy, Pandas
- Matplotlib
- ARCH (GARCH modeling)
- yFinance

## 🚀 How to Run

```bash
pip install -r requirements.txt
python main.py
